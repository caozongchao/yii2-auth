## 安装

推荐使用 composer 安装：

```shell
composer require --prefer-dist yiichina/yii2-auth
```

或者手动编辑 composer.json，添加：

```
"yiichina/yii2-auth": "*"
```

使用文档参考： [Quick Start](https://github.com/yiisoft/yii2-authclient/blob/master/docs/guide/quick-start.md)
